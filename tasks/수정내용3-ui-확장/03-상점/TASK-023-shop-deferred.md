# TASK-023 상점 (P5 — 보류 골격)

> ⏸ **보류(P5).** 2026-06-21 결정으로 **실결제(IAP)는 미도입**(앱인토스 v0.1 결제 제외와 정합). 상점 자체도 전제 시스템(장비·훈련·에너지·확률형) 보류로 함께 보류한다. 도입 시에도 **무과금(골드·다이아)만** 가능하다. 이 문서는 **착수 결정 시 출발점**으로만 유지한다. 현재 코드 변경 없음.

## 보류 사유 / 결정

- **인앱결제 미도입**: 결제 심의·영수증 검증·서버 불필요(MVP "서버 동기화 제외"와 정합).
- **확률형(가챠) 보류**: 드랍 대상(장비·스킬강화 축)이 보류라 대상 자체가 없음. 무료 상자는 도입 시 **확률형이 아닌 정액 지급**으로 설계.
- 보류 상품: 스타터/다이아 팩(실결제), 에너지·스태미나 충전(에너지 미채택), 훈련 티켓(훈련 보류), 장비 상자·전설 조각(장비 보류), 스킬 코어(스킬강화 축 미도입).

## 도입(P5 무과금) 착수 조건

- P3 재화(다이아) 안정화 + 다이아 무료 획득량/사용처 밸런스 확정.
- 무과금 상품 카탈로그(번들·환전·무료 상자) 수치 확정.
- (확률형 도입 시) `docs/game-rating.md` 등급·확률 고지 재검토.

## 도입 시 채울 섹션 (참고: `docs/ui/03-상점.md` §3)

- [ ] 무과금 상품 카탈로그: 무료 상자(매일 1회·정액), 다이아→골드 환전, 골드/다이아 번들. 통화는 골드·다이아만(krw·energy 제외).
- [ ] 일일 무료/한정 리셋: `docs/ui/01-공통-레이아웃.md` 일일 리셋 타이머(다음 로컬 00:00)와 동일 기준, 구매/수령 횟수 저장 추적.
