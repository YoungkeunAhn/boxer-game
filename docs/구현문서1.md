## v0.1에 필요한 기능

- 기본 복서 생성
- 복서 상태 표시
- 샌드백 훈련 실행
- 훈련 시 공격력 증가
- 전투력 계산
- 기본 상대와 경기
- 승리/패배 결과 계산
- 승리 시 돈과 명성 지급
- localStorage 저장
- localStorage 불러오기

## 구현 원칙

- 게임 로직은 UI와 분리한다.
- 전투력 계산, 훈련 실행, 경기 결과 계산은 순수 함수로 작성한다.
- UI 컴포넌트에서는 store의 액션만 호출한다.
- 저장/불러오기는 별도 모듈로 분리한다.
- v0.1에서는 서버 저장을 구현하지 않는다.
- 앱인토스 SDK 연동은 다음 작업으로 분리한다.

## 완료 기준

- 브라우저에서 게임이 실행된다.
- 복서 상태가 화면에 표시된다.
- 훈련 버튼을 누르면 공격력이 증가한다.
- 경기 버튼을 누르면 승리/패배 결과가 표시된다.
- 승리하면 돈과 명성이 증가한다.
- 새로고침 후에도 복서 데이터가 유지된다.
