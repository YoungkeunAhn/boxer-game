# 작업 요청: TASK-005 출시용 리소스 준비 및 검수 체크리스트 정리

현재 복서키우기 v0.1은 핵심 게임 루프 구현, 브라우저 QA, 앱인토스 WebView SDK 연동, 실기기 WebView QA 단계를 진행한 상태다.

이번 작업의 목표는 토스 앱인토스 검수/출시 준비를 위해 필요한 앱 정보, 소개 문구, 리소스 목록, 검수 체크리스트를 정리하는 것이다.

이번 작업에서는 새로운 게임 기능을 추가하지 않는다.

## 이번 작업 목표

복서키우기 v0.1을 앱인토스 제출 준비 단계로 넘기기 위해 필요한 자료와 체크리스트를 정리한다.

## 작업 범위

1. 앱 기본 정보 정리
   - 앱 이름
   - appName
   - 한 줄 소개
   - 짧은 앱 설명
   - 긴 앱 설명
   - 카테고리
   - 고객 문의 이메일
   - 개발자/운영자 정보 TODO

2. 출시용 문구 작성
   - 앱인토스 콘솔 입력용 앱 설명
   - 게임 소개 문구
   - v0.1 기능 소개
   - 플레이 방법 안내
   - 주의사항 문구

3. 이미지 리소스 목록 정리
   - 앱 아이콘
   - 대표 썸네일
   - 스크린샷
   - 게임 로고
   - 기본 캐릭터 이미지
   - 리소스별 필요한 사이즈는 공식 문서 확인 필요로 TODO 처리

4. 검수 체크리스트 정리
   - 앱 실행 여부
   - 첫 화면 정상 표시
   - 훈련/경기/보상 루프 정상 동작
   - 저장/불러오기 정상 동작
   - 재접속 시 데이터 유지
   - 모바일 화면 깨짐 없음
   - 버튼 터치 가능
   - 가로 스크롤 없음
   - 고객 문의 경로 존재
   - 게임 등급분류 준비 상태 확인

5. 게임 등급분류 준비 TODO 정리
   - 게임 설명 자료
   - 주요 화면 스크린샷
   - 플레이 영상
   - 기능 목록
   - 폭력성 표현 수위 설명
   - 결제/광고/확률형 아이템 없음 명시
   - 공식 문서 확인 필요 항목 정리

## v0.1 설명 기준

복서키우기 v0.1은 아래 범위만 설명한다.

포함 기능:

- 복서 생성
- 훈련
- 능력치 성장
- 경기
- 승리/패배 결과
- 보상 획득
- 저장/불러오기
- 재접속 시 진행도 유지

제외 기능:

- 인앱 결제
- 광고
- 리더보드
- 친구 초대
- PVP
- 길드
- 확률형 아이템
- 유료 랜덤박스
- 잔혹한 폭력 표현

## 앱 설명 초안 방향

복서키우기는 무명 복서를 훈련시키고 경기에 도전하며 성장시키는 방치형 복싱 육성 게임이다.

플레이어는 훈련을 통해 공격력, 체력, 방어력, 스피드를 올리고, 더 강한 상대에게 도전해 돈과 명성을 얻는다.

v0.1에서는 복잡한 결제나 광고 없이 기본 성장 루프를 중심으로 플레이 경험을 검증한다.

## 수정 원칙

- 실제 구현되지 않은 기능은 설명에 넣지 않는다.
- 결제, 광고, 랭킹, 친구 초대는 v0.1 설명에서 제외한다.
- 등급분류나 정책 관련 내용은 단정하지 말고 공식 문서 확인 필요로 표시한다.
- 앱인토스 콘솔에 바로 옮겨 적을 수 있는 문구로 작성한다.
- 과장된 표현보다 실제 기능 중심으로 작성한다.
