# 파이터 — 메인 전투 화면

홈/기본 탭. 출처 이미지: `메인ui1.png`, `메인ui2.png`. 두 이미지의 차이는 **파이터 타입 전환 패널 유무**(메인ui1에 있음).

## 1. 이미지에서 확인된 요소

### 상단 전투 헤더 (상단 바 아래)
```text
[🗺 월드맵]   STAGE 12-3  ●─●─●─◇─◇(보스)   [👹 보스 마커들]
[복서 카드: 아웃파이터 남자 | HP 7,850/8,450 | 🔥3.25K]   VS   [몬스터 카드: 늑대 복서 | 15,420/22,500 | 🔥2.80K]
                              ⚔ 자동 전투 중
```
- **월드맵 버튼**(좌상단) — 신규(스테이지/챕터 선택 화면 진입 추정).
- **스테이지 진행 인디케이터**: `STAGE 12-3` + 5칸 점(일반 4 + 보스 1) + 보스 마커.
- **대결 카드 2개**: 좌(복서) / 우(몬스터). 각 카드에 **이름 · 현재HP/최대HP · 공격력(🔥)**.
- "자동 전투 중" 상태 텍스트.

### 중앙 전투 스테이지
- 링 배경 + 복서/몬스터 스프라이트 대치.
- **데미지 플로팅 숫자**(-1,245 / -887, 치명타는 큰 글씨).
- 타격 이펙트(스파크).

### 전투 컨트롤 (우측/중앙 하단)
- **AUTO ON 토글** — 자동 전투 on/off.
- **스킬 발동 버튼**(원형, 쿨다운 12.3초 표시) — **수동 스킬 발동** 신규.
- **배속 토글 ▶▶ x2** — 전투 속도 신규.

### (메인ui1 전용) 파이터 타입 전환 패널
```text
현재 파이터 타입: 아웃파이터 남자
[인파이터 남자] [인파이터 여자] [아웃파이터 남자✓] [아웃파이터 여자]
```
- 4종 타입/성별 카드 가로 스크롤, 현재 선택 강조.

### 강화 패널 (하단 그리드 2×3)
```text
공격력 Lv.125  🔥3.25K +25     체력 Lv.118  ❤8,450 +120     속도 Lv.96  👟1.32 +0.02
스킬강화 Lv.82 스킬뎀+82%       장비강화 Lv.74 장비효과+74%    훈련강화 Lv.66 훈련효과+66%
각 칸 하단: [🪙 비용] 버튼 (12.4K / 10.8K / 8,160 / 6,200 / 5,100 / 4,200)
```
- 각 칸: 아이콘 · 이름 · 레벨 · 현재값 · **다음 레벨 증가량(+표시)** · 비용 버튼.
- 강화 카테고리 6종: **공격력 / 체력 / 속도 / 스킬강화 / 장비강화 / 훈련강화**.

## 4. 범위·커플링 플래그

- 🟡 헤더/카드/진행바는 표시 위주 — 전부 기존 셀렉터·`stages.ts`/`formulas.ts`에서 파생(신규 로직 없음).
- 🔴 AUTO/수동스킬/배속은 **전투 로직·입력 변경** → `combat.ts`/`gameStore.ts` 타이머·`resolveAttack`·`advanceCombat` 영향. 밸런스 수치는 불변(보스 타임아웃 게임시간 기준) → `BALANCE_VERSION` 불필요, 단 회귀 테스트·`npm run e2e` 필수.
- 🟢 강화 9종 유지 → 수식·저장·`BALANCE_VERSION` 변경 없음. 표현(그룹/탭)만 조정.
- 🟡 타입 외형 전환은 `boxer.type`/`gender` 변경 경로 추가(신규 저장 필드 없음). 스킬 슬롯·`typeMultiplier` 재적용 정합 확인.
