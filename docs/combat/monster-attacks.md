# 몬스터 공격 시스템

## 핵심

몬스터도 복서를 공격한다. 몬스터 공격은 복서 타입의 차이를 보여주는 장치다. → [복서 타입](../boxer/types.md)

가정: 몬스터 공격은 `수정내용2`에서 도입된 신규 시스템이다. 기존 구현은 몬스터가 복서를 공격하지 않았으므로, 적용 시 [체력과 실패](./hp-and-defeat.md)·[데이터 모델](../systems/data-model.md)·밸런스 버전을 함께 갱신한다.

## 기본 흐름

```text
몬스터 공격 쿨타임 완료
→ 공격 예고
→ 공격 발동
→ 복서 회피/가드 판정
→ 회피 성공: MISS + 카운터
→ 가드 성공: 피해 감소
→ 실패: 복서 HP 감소
```

## 몬스터 공격 타입

```text
기본 공격 / 강공격 / 연속 공격 / 상태이상 공격 / 보스 패턴 공격
```

## 타입별 반응

| 복서 타입 | 반응 |
| --- | --- |
| 인파이터 | GUARD! → 피해 감소 → 근접 반격 |
| 아웃복서 | STEP BACK! → MISS! → COUNTER STRAIGHT! |

- 회피율은 아웃복서가 높고, 가드·피해 감소는 인파이터가 높다.
- 회피 성공은 카운터 기회로, 가드 성공은 피해 감소로 이어진다.
- 회피·가드에 모두 실패하면 복서 HP가 감소한다.

## 판정 수치 (확정값 balanceVersion 9, TASK-006)

아래 값은 전부 `constants.ts`의 확정 상수와 동일하다(TASK-013 동결). 판정은
**회피 → 가드 → 피격** 순으로 한 번의 공격에 배타적으로 적용한다. 카운터는 회피(아웃복서)·가드(인파이터)
성공의 부수효과로만 발생하며 단독 분기가 아니다.

| 항목 | 값 | 비고 |
| --- | --- | --- |
| 기본 회피율 | 5% | 강화 레벨당 +1%p, 상한 60% |
| 카운터 기본 계수 | 1.0 | 강화 레벨당 +0.04 |
| 가드 추가 피해 감소 | 25% | 방어 유도 감소율과 합산, 합산 상한 90% |
| 회피 카운터 데미지율 | 0.8 | 기대 타격 피해 × counter 계수 × 0.8 |
| 인파이터 가드 반격 데미지율 | 0.3 | 카운터보다 약하게 |

- 결과 분류: `HIT`(감소 없이 피격) / `GUARD`(가드 감소 적용 피격) / `MISS`(회피, 피해 0) / `COUNTER`(회피+반격).
- 가정: 카운터·가드 반격은 몬스터를 처치하지 않고 `monsterHp`만 깎는다. 보스 그로기 누적은 [보스전](./boss.md)(TASK-009)에서 처리한다.
- 타입 보정: 인파이터 회피 ×0.6·카운터 ×0.5·가드 피해감소 ×1.2, 아웃복서 회피 ×1.6·카운터 ×1.6·가드 피해감소 ×0.9.
