# 체력과 실패 구조

## 핵심

복서에게 HP가 있다. 복서 HP가 0이 되면 스테이지 실패다. 하지만 게임오버가 아니라 현재 스테이지에 머문다.

가정: 복서 HP와 피격은 `수정내용2`에서 도입된 신규 시스템이다. 기존 MVP는 플레이어 HP가 없었으므로([MVP 범위](../overview/mvp-scope.md) 참고) 적용 시 저장 스키마·밸런스 버전을 갱신한다.

## 실패 처리

```text
Stage 2-4 울프 대장 도전
복서 HP 0
→ KNOCK DOWN!
→ Stage 2-4 유지
→ 골드 일부 획득
→ 업그레이드 후 재도전
```

- HP가 0이 되어도 진행 위치는 잃지 않는다.
- 실패 시 현재 스테이지에 머문 채 골드를 일부 확보하고 재도전한다.
- 보스 실패 처리는 [보스전](./boss.md)을 따른다.

## 성장 루프

```text
막힘 → 돈 모음 → 체력/방어/스킬 강화 → 재도전 → 스테이지 돌파
```

## 기존 루프와의 관계

- 기존 보스 실패 루프(직전 일반 스테이지 반복 파밍)는 유지한다. → [핵심 루프](../overview/core-loop.md)
- HP 도입으로 일반 스테이지에서도 실패가 가능해지며, 회피/가드/방어 강화가 생존 수단이 된다. → [몬스터 공격](./monster-attacks.md)
