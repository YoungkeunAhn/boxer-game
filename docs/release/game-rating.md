# 게임 등급분류 준비

## 문서 목적

복서키우기 자동 전투 MVP의 콘텐츠를 자체 점검하고 실제 등급분류 또는 심의 자료를 준비한다. 이 문서는 법률 자문이나 등급 판정을 대신하지 않는다.

## 검토가 필요한 이유

- 복서가 몬스터를 반복 타격하는 게임이므로 비현실적 표현이어도 폭력성 노출을 검토해야 한다.
- 앱인토스 입점 정책과 유통 경로에 따라 절차가 달라질 수 있다. (`공식 문서 확인 필요`)
- 기능과 연출이 바뀌면 판단이 달라질 수 있으므로 제출 직전 최종 빌드로 다시 점검한다.

## 확인할 등급분류 경로

- 게임물관리위원회 등급분류 대상 및 신청 절차 (`공식 문서 확인 필요`)
- 앱스토어·플레이스토어·원스토어 자체등급분류 적용 조건 (`공식 문서 확인 필요`)
- 앱인토스가 요구하는 등급 정보, 증빙과 표시 방식 (`공식 문서 확인 필요`)

## 콘텐츠 리스크 점검

| 항목 | v0.1 방향 | 확인 사항 |
| --- | --- | --- |
| 자동 전투 | 복서와 판타지 몬스터의 반복 전투 | 타격 빈도와 장시간 노출 수위 |
| 보스 | 30초 제한 시간 내 공격 | 긴박한 연출이 과격하지 않은지 |
| 폭력 표현 | 밝고 비현실적이며 절제된 연출 | 출혈·부상·잔혹·사망 표현 미포함 |
| 치명타 | 전투 피해 배율만 확률 판정 | 현금 가치·베팅·확률형 보상과 연결하지 않음 |
| 보상 | 처치마다 수식으로 정해지는 골드 | 사행성 오인 요소 배제 |
| 결제·광고 | 미포함 | 버튼, 문구와 SDK도 제외 |
| 이용자 데이터 | 기기 로컬 저장 | 수집·전송 여부와 고지 필요성 |

## 폭력성과 사행성 관련 원칙

- 몬스터 타격은 과격하거나 현실적인 상해로 묘사하지 않는다.
- 출혈, 상처, 골절, 신체 훼손, 사망과 고통을 강조하는 표현은 제외한다.
- 치명타는 무료 전투 계산이며 획득 확률형 아이템이나 유료 재화와 연결하지 않는다.
- 처치 골드와 강화 비용은 공개된 고정 수식으로 계산한다.
- 보스 실패를 조롱하거나 잔혹한 연출로 보상하지 않는다.
- 추후 결제, 광고나 랜덤 보상을 추가하면 등급·고지·청소년 보호 요건을 다시 검토한다. (`공식 문서 확인 필요`)

## 개인정보 및 계정 저장

- v0.1은 localStorage에 복서 이름과 게임 진행도만 저장하고 불필요한 개인정보를 수집하지 않는다.
- 앱인토스 사용자 식별자, 계정 연동과 서버 저장은 제외한다.
- SDK가 제공하거나 전송하는 식별 정보와 로그 범위를 확인한다. (`공식 문서 확인 필요`)
- 문의 과정에서 개인정보를 받는 경우 보관·파기·고지 방식을 확인한다. (`공식 문서 확인 필요`)

## 준비 자료

- 자동 공격 → 처치 골드 → 강화 → 보스 → 반복 파밍의 전체 기능 설명
- 시작부터 보스 성공·실패와 재도전까지의 플레이 영상
- 일반 몬스터, 보스, 치명타, 처치와 강화 화면 스크린샷
- 캐릭터, 몬스터, 효과음과 타격 연출 목록
- 결제·광고·랜덤박스·유료 확률형 아이템 미포함 설명
- 저장 데이터 항목과 외부 전송 여부 설명
- 제출 버전과 자료 버전의 일치 기록
