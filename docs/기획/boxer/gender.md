# 캐릭터 성별 구조

## 핵심

성별은 타입을 대체하지 않는다. 전투 성능의 핵심 차이는 인파이터/아웃복서 타입이 만든다. → [복서 타입](./types.md)

성별은 다음 요소에만 영향을 준다.

```text
외형 / 실루엣 / 헤어 / 의상 / 동작 느낌 / 스킬 모션의 무게감
```

## 인파이터 남자

묵직하고 단단한 탱커형 이미지.

```text
외형: 단단한 상체, 낮은 자세, 두꺼운 팔과 어깨, 운동복, 짧은 검정 헤어
동작: 묵직한 펀치, 단단한 가드, 잘 밀리지 않음, 강한 훅/어퍼
```

## 인파이터 여자

작지만 단단하고 빠르게 파고드는 압박형 이미지.

```text
외형: 작고 민첩, 낮은 자세, 강한 눈매, 운동복, 검정 헤어, 간결함
동작: 짧은 거리에서 빠르게 파고듦, 날카로운 리버샷, 좋은 연속 압박
```

## 아웃복서 남자

가볍게 움직이고 거리를 유지하는 카운터형 이미지.

```text
외형: 길고 가벼운 실루엣, 큰 상체 아님, 강조된 발 움직임, 운동복, 검정 헤어
동작: 자연스러운 스텝백, 길게 뻗는 스트레이트, 시원한 회피 후 카운터
```

## 아웃복서 여자

빠르고 날렵한 회피/카운터형 이미지.

```text
외형: 가벼운 실루엣, 빠른 발놀림, 민첩한 자세, 운동복, 검정 헤어
동작: 잘 어울리는 고스트스텝, 자연스러운 나비 스텝, 빠른 팬텀 잽
```
