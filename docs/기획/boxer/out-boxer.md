# 아웃복서

거리를 두고 피한 뒤 강하게 카운터치는 회피/카운터형 복서다.

## 특징

```text
체력 낮음
방어력 낮음
회피율 높음
카운터 데미지 높음
스트레이트, 스텝백, 고스트스텝, 나비 스텝과 잘 어울림
```

## 역할

```text
맞지 않는다.
거리 조절로 몬스터 공격을 흘린다.
회피 후 카운터로 큰 데미지를 준다.
```

## 전투 느낌

```text
몬스터 공격
→ STEP BACK!
→ MISS!
→ COUNTER STRAIGHT! → GHOST STEP! → PHANTOM JAB!
```

## 강점과 약점

| 강점 | 약점 |
| --- | --- |
| 회피·스텝백 | 체력 낮음 |
| 거리 조절 | 방어 낮음 |
| 카운터·스트레이트 | 회피 실패 시 위험 |
| 쿨타임 가속 | |

## 주요 스킬

기본 스킬(잽/스트레이트/훅/어퍼)에 더해 전용 스킬로 회피·카운터 정체성을 강화한다. → [아웃복서 전용 스킬](../skills/out-boxer-skills.md)

```text
고스트스텝 / 나비 스텝 / 스텝백 카운터 / 팬텀 잽 / 거리 조절
```

## 전투 키워드

```text
MISS / STEP BACK / GHOST STEP / COUNTER STRAIGHT / DISTANCE CONTROL
```
