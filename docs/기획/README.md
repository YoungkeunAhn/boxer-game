# 복서키우기

무명 복서를 훈련시켜 더 높은 리그와 챔피언에 도전하는 방치형 복싱 육성 게임이다.

## 핵심 루프

`훈련 → 능력치 상승 → 경기 → 보상 획득 → 업그레이드 → 더 강한 상대 도전`

## MVP 범위

- 포함: 복서 생성/조회, 훈련과 성장, 경기 판정, 보상, 장비, 자동 훈련, 저장/불러오기
- 제외: PVP, 길드, 랭킹, 시즌패스, 결제, 광고, 복잡한 스킬, 스토리 모드
- 완료 기준: 신규 저장 데이터로 시작해 자동 훈련 해금 후 재접속 복구까지 한 흐름으로 동작한다.

## 추천 읽기 순서

1. 프로젝트 개요와 핵심 루프로 게임 방향을 이해한다.
2. MVP 범위와 유저 플로우로 첫 버전의 경계를 확인한다.
3. 게임 시스템, 능력치와 수식, 콘텐츠 데이터로 규칙을 구체화한다.
4. 데이터 모델과 개발 로드맵을 기준으로 구현한다.
