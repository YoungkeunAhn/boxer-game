# 게임 핵심 방향과 한 줄 정의

## 핵심 방향

복서키우기는 클릭커 게임이 아니다. 복서가 자동으로 몬스터와 전투하고, 플레이어는 골드와 성장 재화로 복서의 능력치·스킬·타입별 특성을 강화하는 방치형 복싱 RPG다.

## 핵심 전투 구조

```text
스테이지 입장
→ 복서 자동 공격
→ 몬스터 공격
→ 회피 / 가드 / 카운터 판정
→ 몬스터 처치 시 다음 스테이지
→ 복서 HP 0이면 현재 스테이지 유지
→ 골드 획득
→ 업그레이드 후 재도전
```

이 구조는 [핵심 루프](./core-loop.md), [몬스터 공격](../combat/monster-attacks.md), [체력과 실패](../combat/hp-and-defeat.md)에서 상세히 다룬다.

## 한 줄 정의

```text
복싱 스킬 쿨타임 자동전투를 기반으로,
인파이터와 아웃복서의 전투 스타일을 선택하고,
성별 캐릭터 외형과 타입별 스킬을 성장시키며,
몬스터와 보스를 돌파하는 방치형 복싱 RPG
```
