# 콘텐츠 데이터

아래 값은 구현 형식을 검증하기 위한 초안이다. 가정: 모든 수치는 플레이테스트 전 임시값이며 확정 시 `balanceVersion`을 올린다.

| `jump_rope` | 줄넘기 | 스피드 +2 | 행동력 1 | 기본 |

## 데이터 검증 규칙

- ID는 영문 소문자 `snake_case`이며 종류 안에서 유일해야 한다.
- 가격, 보상, 능력치, 배율은 음수가 될 수 없다.
- 참조하는 해금 조건과 효과 키가 실제 데이터에 존재해야 한다.
