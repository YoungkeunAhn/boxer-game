# MVP 범위

## v0.1 목표

신규 플레이어가 복서를 만들고 훈련과 경기를 반복해 장비 및 자동 훈련을 해금한 뒤, 저장 데이터를 복구할 수 있는 단일 성장 루프를 검증한다.

## 포함 기능

- 복서 생성: 이름 입력, 초기 능력치 설정
- 복서 상태: 능력치, 전투력, 돈, 명성 표시
- 훈련: 종류 선택, 비용 검사, 능력치 증가
- 경기: 상대 선택, 승률 계산, 결과 판정
- 보상: 승리 시 돈과 명성 지급
- 장비: 목록, 가격, 구매, 장착 효과 적용
- 자동 훈련: 조건 확인, 해금, 자동 성장
- 저장/불러오기: 진행 상태 저장, 재접속 시 복원
- 오프라인 보상: 마지막 저장 시각 기준 자동 훈련 보상

## 제외 기능

- PVP, 길드, 랭킹
- 시즌패스, 결제, 광고
- 복잡한 액티브/패시브 스킬
- 스토리 모드와 분기형 대화
- 실시간 복싱 조작 및 물리 기반 전투
- 서버 계정 동기화와 부정행위 방지

## 완료 기준

- 신규 저장 데이터에서 오류 없이 복서를 생성할 수 있다.
- 모든 기본 훈련이 올바른 능력치를 증가시킨다.
- 표시 승률과 동일한 규칙으로 경기 결과가 판정된다.
- 경기 보상이 중복 없이 지급되고 구매 시 잔액과 효과가 반영된다.
- 자동 훈련 해금 전후 상태와 오프라인 보상이 정상 동작한다.
- 저장 후 앱을 다시 열어 핵심 진행 상태가 복원된다.
- 잘못되거나 구버전인 저장 데이터에 안전한 기본값을 적용한다.
- 핵심 산식과 저장 마이그레이션에 자동 테스트가 있다.

## 수용 시나리오

1. 플레이어가 복서 이름을 정하고 상태 화면에 진입한다.
2. 샌드백 훈련으로 공격력을 높여 첫 상대 승률을 개선한다.
3. 첫 경기에서 승리해 돈과 명성을 받는다.
4. 글러브를 구매해 전투력 증가를 확인한다.
5. 조건을 충족해 자동 훈련을 해금한다.
6. 게임 종료 후 재접속하여 저장 상태와 오프라인 보상을 확인한다.
