# 인파이터 전용 스킬

인파이터는 전용 스킬로 탱커형 압박 복서의 느낌을 강화한다. → [인파이터](../boxer/infighter.md)

가정: 아래 효과·수치는 `수정내용2` 설계 방향이며 밸런스는 미확정이다.

## 리버샷

바디를 파고드는 고급 공격. 인파이터 대표 공격이다.

```text
역할: 내상, 지속 피해, 보스 그로기 증가
예시: LIVER SHOT! -160 / 내상 발생(5초 추가 피해) / Groggy +20
```

## 철벽가드 (패시브)

몬스터 강공격을 맞아도 버티는 방어 패시브.

```text
역할: 피해 감소, 생존력 증가, 탱커성 강화
예시: IRON GUARD! 받는 피해 -60%
```

## 압박

가까이 붙어 몬스터를 몰아붙이는 버프.

```text
역할: 훅/어퍼 데미지 증가, 몬스터 공격력 감소
예시: PRESSURE! 6초간 훅/어퍼 +20%, 몬스터 공격력 -10%
```

## 가젤펀치

앞으로 뛰어들며 강하게 치는 고급 공격.

```text
역할: 거리 좁히기, 강한 단일 피해, 넉백, 그로기 증가
예시: GAZELLE PUNCH! CRIT -240 / Groggy +15
```

## 뎀프시롤

인파이터의 대표 필살기. 좌우로 흔들며 연속 훅을 날린다.

```text
역할: 연속 타격, 보스 폭딜, 그로기 대량 증가
예시: DEMPSEY ROLL! LEFT HOOK -80 / RIGHT HOOK -92 / LEFT HOOK CRIT -130 / RIGHT HOOK -110
```
