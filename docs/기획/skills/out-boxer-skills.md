# 아웃복서 전용 스킬

아웃복서는 전용 스킬로 회피/거리 조절/카운터의 느낌을 강화한다. → [아웃복서](../boxer/out-boxer.md)

가정: 아래 효과·수치는 `수정내용2` 설계 방향이며 밸런스는 미확정이다.

## 고스트스텝

순간적으로 사라지듯 회피하고 카운터를 준비하는 특수 회피기.

```text
역할: 완전 회피, 카운터 데미지 증가, 스트레이트 쿨타임 초기화 가능
예시: GHOST STEP! MISS! → COUNTER STRAIGHT! CRIT -210
```

## 나비 스텝

가볍게 움직이며 회피율과 쿨타임 회복 속도를 올리는 풋워크 버프.

```text
역할: 회피율 증가, 스킬 쿨타임 회복 속도 증가, 카운터 확률 증가
예시: NAVI STEP! 5초간 회피율 +15%, 쿨타임 회복 +20%, 카운터 확률 +10%
```

## 스텝백 카운터 (패시브)

뒤로 빠져 피한 뒤 스트레이트로 반격하는 아웃복서 대표 카운터.

```text
역할: 회피 후 강한 반격, 스트레이트 강화, 보스 강공격 대응
예시: STEP BACK! MISS! → COUNTER STRAIGHT! CRIT -180
```

## 팬텀 잽

빠르게 잽을 여러 번 넣고 빠지는 고급 공격.

```text
역할: 빠른 다단 잽, 짧은 회피율 증가, 안정 딜링
예시: PHANTOM JAB! JAB -24 / JAB -24 / JAB -28
```

## 거리 조절

몬스터의 공격 템포를 늦추는 아웃복서형 버프.

```text
역할: 몬스터 공격 쿨타임 증가, 복서 회피율 증가, 카운터 기회 증가
예시: DISTANCE CONTROL! 6초간 몬스터 공격 쿨타임 +20%, 복서 회피율 +10%
```
