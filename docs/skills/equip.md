# 스킬 장착 구조

## 핵심

공통 기본 스킬은 항상 사용한다. 타입 전용 스킬은 장착 슬롯으로 제한한다.

```text
공통 기본 스킬 4개 = 항상 자동 사용
타입 전용 스킬   = 액티브 슬롯 최대 3개 장착
패시브 스킬      = 패시브 슬롯 1개 적용
```

가정(TASK-010 구현, `constants.ts`와 동기화):

- 슬롯 수: 액티브 `ACTIVE_SKILL_SLOT_MAX = 3`, 패시브 `PASSIVE_SKILL_SLOT_MAX = 1`.
- 발동 정책: 액티브 스킬은 **쿨타임이 끝나면 자동 발동**하며, 한 틱에 발동 가능한 스킬이 여럿이면 **슬롯 순서(Slot1 > Slot2 > Slot3)**로 1개를 발동한다. 기본 타격은 그대로 나가고 스킬은 그 위에 별도로 더해진다.
- 첫 발동: 전투 시작 시 모든 액티브 스킬은 `cooldownMs`가 지난 뒤 처음 발동한다(시작 즉시 발동 아님).
- 기본 장착(`DEFAULT_EQUIPPED_SKILLS`): 인파이터 = 리버샷·압박·뎀프시롤 + 철벽가드, 아웃복서 = 팬텀 잽·고스트스텝·나비 스텝 + 스텝백 카운터.
- 해금 조건·획득 경제는 미확정(TODO). 초기엔 타입별 기본 제공만 한다.

## 슬롯 구성

| 슬롯 | 내용 |
| --- | --- |
| 기본 스킬 | 잽 / 스트레이트 / 훅 / 어퍼 (고정, 항상 사용) |
| 전용 슬롯 1~3 | 타입 전용 액티브 스킬 |
| 패시브 슬롯 | 타입 전용 패시브 스킬 |

기본 스킬은 [공통 기본 공격](../combat/basic-attacks.md)을 참고한다.

## 인파이터 장착 예시

```text
기본:   잽 / 스트레이트 / 훅 / 어퍼
전용:   Slot 1 리버샷 / Slot 2 압박 / Slot 3 뎀프시롤
패시브: 철벽가드
```

→ [인파이터 전용 스킬](./infighter-skills.md)

## 아웃복서 장착 예시

```text
기본:   잽 / 스트레이트 / 훅 / 어퍼
전용:   Slot 1 팬텀 잽 / Slot 2 고스트스텝 / Slot 3 나비 스텝
패시브: 스텝백 카운터
```
