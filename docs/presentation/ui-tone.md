# 타입별 UI 톤

UI 톤은 복서 타입의 느낌을 강조한다. 공통 레이아웃은 [UI 구조](./ui.md)를 따른다.

## 인파이터 UI

묵직하고 단단한 느낌.

```text
강한 타격 이펙트
피해 감소 표시
GUARD 텍스트 강조
훅/어퍼 이펙트 강조
붉은 압박감
무거운 화면 흔들림
```

예시:

```text
IRON GUARD! 피해 -60%
LIVER SHOT! 내상 발생
DEMPSEY ROLL!
```

## 아웃복서 UI

빠르고 가벼운 회피/카운터 느낌.

```text
잔상 효과
빠른 스텝 이펙트
MISS 텍스트 강조
COUNTER 텍스트 강조
스트레이트 궤적 강조
가벼운 이동감
```

예시:

```text
GHOST STEP! MISS!
COUNTER STRAIGHT! CRIT -210
```
