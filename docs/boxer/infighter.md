# 인파이터

가까이 붙어서 맞아도 버티며 계속 압박하는 탱커형 복서다.

## 특징

```text
체력 높음
방어력 높음
피해 감소 높음
회피율 낮음
카운터 성능 낮음
훅, 어퍼, 리버샷, 뎀프시롤과 잘 어울림
```

타입 보정 계수(확정값 balanceVersion 10, `constants.ts`와 동일) — 체력 ×1.3 / 방어 ×1.3 / 가드 피해감소 ×1.2 / 회피 ×0.6 / 카운터 ×0.5 / 그로기 누적 ×1.4. 가드 성공 시 약한 근접 반격(기대 타격 피해 × counter 계수 × 0.3)을 한다.

## 역할

```text
맞으면서 버틴다.
가까이 붙어서 몰아붙인다.
바디 공격과 그로기로 보스를 무너뜨린다.
```

## 전투 느낌

```text
몬스터 공격
→ GUARD!
→ 피해 감소
→ JAB! → HOOK! → LIVER SHOT! → DEMPSEY ROLL!
```

## 강점과 약점

| 강점 | 약점 |
| --- | --- |
| 체력·방어·피해 감소 | 회피 낮음 |
| 훅/어퍼 화력 | 카운터 낮음 |
| 리버샷·뎀프시롤 폭딜 | 공격을 많이 맞음 |
| 그로기 누적 | |

## 주요 스킬

```text
리버샷 / 철벽가드 / 압박 / 가젤펀치 / 뎀프시롤
```
