# 복서 타입

복서는 전투 스타일에 따라 두 타입으로 나뉜다. 타입이 전투 성능의 핵심 차이를 만든다.

```text
인파이터 = 탱커형 압박 복서
아웃복서  = 회피/카운터형 복서
```

타입과 스킬은 `수정내용2`로 구현 완료됐고 밸런스 수치는 TASK-013에서 확정·동결했다(balanceVersion 10). 수치를 바꾸면 [능력치와 수식](../systems/stats-and-formulas.md)과 밸런스 버전을 함께 갱신한다.

## 타입 선택

- 각 타입은 남자/여자 캐릭터를 모두 가진다(총 4종).
- 성별은 외형·실루엣·모션 무게감 차이만 주고, 전투 성능 차이는 타입이 결정한다. → [캐릭터 성별](./gender.md)

```text
인파이터 남자 / 인파이터 여자
아웃복서 남자 / 아웃복서 여자
```

## 타입 비교

| 항목 | 인파이터 | 아웃복서 |
| --- | --- | --- |
| 역할 | 탱커형 압박 | 회피/카운터 |
| 체력 | 높음 | 낮음 |
| 방어력·피해 감소 | 높음 | 낮음 |
| 회피율 | 낮음 | 높음 |
| 카운터 성능 | 낮음 | 높음 |
| 대표 스킬 | 리버샷, 뎀프시롤 | 고스트스텝, 스텝백 카운터 |
| 핵심 키워드 | GUARD, PRESSURE, GROGGY | MISS, COUNTER, STEP BACK |

## 전투 흐름에서의 차이

타입은 몬스터 공격에 대한 반응 방식에서 가장 크게 드러난다. → [몬스터 공격](../combat/monster-attacks.md)

```text
인파이터: 몬스터 공격 → GUARD! → 피해 감소 → 근접 반격
아웃복서: 몬스터 공격 → STEP BACK! → MISS! → COUNTER STRAIGHT!
```
